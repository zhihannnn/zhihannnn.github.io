Selected Projects

Enterobacter-TAOG-Finder

Python toolkit for T6SS-associated ortholog group discovery in Enterobacter genomes

Enterobacter-TAOG-Finder is a computational project developed for large-scale comparative genomic analysis of Type VI secretion systems in Enterobacter. The workflow supports T6SS locus detection, ortholog-group presence/absence matrix construction, P/A-value calculation, and genome-context-based identification of T6SS-associated genes.

View code and example datasets on GitHub

Python Reimplementation of Weighted Nearest Neighbor Analysis

Method reproduction project for multimodal single-cell RNA and ADT data integration

This project focuses on reproducing key components of the Seurat v4 Weighted Nearest Neighbor workflow for multimodal single-cell data analysis. The implementation reconstructs major steps including modality-specific preprocessing, nearest-neighbor graph construction, Jaccard/SNN-based bandwidth estimation, cell-specific RNA/ADT modality weighting, weighted multimodal neighbor graph construction, Leiden clustering, and UMAP visualization.

Through this project, I explored multimodal representation learning, graph-based integration, adaptive modality weighting, and reproducible Python/R workflows for single-cell RNA and protein data analysis. The Python implementation was cross-checked against manually exported R/Seurat intermediate results to better understand the WNN algorithm step by step.

View project code on GitHub

The first plot shows cell-type-specific RNA modality weights, while the second plot compares WNN-based Leiden clustering with reference cell-type annotations.

Status: Ongoing methodological reproduction project.
